## 🎯 평가 기준 (각 항목 1-10점)

### 1. 신학적 정확성 (가중치 30%)
**평가 항목:**
- 성경 인용의 정확성
- 신학적 오류 없음
- 교리적 적합성
- 성경 해석의 적절성

**평가 시 확인사항:**
- 성경 구절이 정확하게 인용되었는가?
- 성경 해석이 문맥에 맞는가?
- 이단적이거나 잘못된 교리가 없는가?
- 복음의 핵심 메시지가 담겨있는가?

**점수 가이드:**
- 9-10점: 신학적으로 완벽하고 깊이 있음
- 7-8점: 정확하며 적절함
- 5-6점: 대체로 괜찮으나 일부 개선 필요
- 1-4점: 심각한 신학적 오류 또는 부정확함

### 2. 콘텐츠 구조 (가중치 25%)
**평가 항목:**
- 도입-본론-결론 구조
- 문단 구성의 적절성
- 논리적 흐름
- 섹션 간 연결성

**평가 시 확인사항:**
- 각 섹션(성경 소개, 묵상, 찬송가, 적용, 기도)이 모두 있는가?
- 소주제(H3)가 3-4개로 적절히 나뉘어져 있는가?
- 문단 간 논리적 연결이 자연스러운가?
- 전체 구조가 읽기 쉬운가?

**점수 가이드:**
- 9-10점: 구조가 완벽하고 흐름이 탁월함
- 7-8점: 구조가 잘 짜여있음
- 5-6점: 기본 구조는 있으나 개선 필요
- 1-4점: 구조가 혼란스럽거나 누락됨

### 3. 독자 참여도 (가중치 20%)
**평가 항목:**
- 독자 공감 유도
- 실생활 적용 가능성
- 감정적 연결
- 구체적 예시 사용

**평가 시 확인사항:**
- 일상적 질문이나 상황으로 시작하는가?
- "오늘의 적용" 항목이 구체적이고 실천 가능한가?
- 독자가 자신의 삶과 연결할 수 있는가?
- 따뜻하고 진솔한 톤을 유지하는가?

**점수 가이드:**
- 9-10점: 독자와의 공감대 형성이 탁월함
- 7-8점: 공감되고 적용 가능함
- 5-6점: 기본적인 공감 요소는 있음
- 1-4점: 추상적이고 공감하기 어려움

### 4. 기술적 품질 (가중치 15%)
**평가 항목:**
- 맞춤법 및 문법
- 적절한 어휘 사용
- 문장 길이의 적절성
- 가독성
- **유튜브 임베딩과 성경 구절 일치 여부**

**평가 시 확인사항:**
- 맞춤법 오류가 없는가?
- 문법이 정확한가?
- 복잡하거나 어려운 용어를 피했는가?
- 문장이 너무 길거나 짧지 않은가?
- 이모지를 사용하지 않았는가? (금지사항)
- **유튜브 영상이 임베딩되어 있다면, 영상 제목이 본문에서 다루는 성경 구절과 일치하는가?**
- **유튜브 영상의 성경책명/장이 본문의 성경 구절과 같은가?** (예: 에베소서 2장 블로그라면 유튜브도 에베소서 2장이어야 함)

**점수 가이드:**
- 9-10점: 기술적으로 완벽함 + 유튜브 영상이 성경 구절과 정확히 일치
- 7-8점: 오류가 거의 없음 + 유튜브 영상이 적절함
- 5-6점: 일부 오류 있으나 읽을 만함 또는 유튜브 영상이 성경 구절과 불일치
- 1-4점: 많은 오류로 읽기 어려움 또는 유튜브 영상이 완전히 무관함

### 5. SEO 최적화 (가중치 10%)
**평가 항목:**
- 키워드 자연스러운 포함
- 메타 설명 최적화
- 제목의 매력도
- 구조적 마크업 (H1, H2, H3)

**평가 시 확인사항:**
- 키워드가 제목과 본문에 자연스럽게 포함되었는가?
- 메타 설명이 150-160자로 적절한가?
- 제목이 검색과 클릭을 유도하는가?
- H1, H2, H3 헤딩 구조가 올바른가?

**점수 가이드:**
- 9-10점: SEO 최적화가 완벽함
- 7-8점: SEO 요소가 잘 반영됨
- 5-6점: 기본적인 SEO는 되어있음
- 1-4점: SEO 고려가 부족함

## 📌 평가 지침

### 종합 점수 계산
total_score = (theological_accuracy × 0.3) + (content_structure × 0.25) + (engagement × 0.2) + (technical_quality × 0.15) + (seo_optimization × 0.1)

### 발행 권장사항 (recommendation)
- **"publish"**: total_score >= 7.0 이고 치명적 문제 없음
- **"revise"**: 5.0 <= total_score < 7.0 또는 일부 개선 필요
- **"reject"**: total_score < 5.0 또는 심각한 신학적 오류

### 신뢰도 (confidence)
- **"high"**: 모든 평가 항목이 명확함
- **"medium"**: 일부 애매한 부분 존재
- **"low"**: 평가가 어려운 요소 다수

### 치명적 문제 (critical_issues)
다음 사항이 발견되면 반드시 critical_issues에 기록:
- 심각한 신학적 오류
- 이단적 교리
- 성경 왜곡
- 부적절한 표현
- 사실과 다른 정보
- **유튜브 임베딩 영상의 성경 구절이 본문과 완전히 다른 경우** (예: 본문은 에베소서 2장인데 유튜브는 창세기 1장)
