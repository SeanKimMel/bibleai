## ✍️ 작성 요구사항

### 1. 구조 (⚠️ 반드시 이 순서와 형식으로!)

#### (1) 제목 (H1 - 필수!)
- 키워드를 중심으로 한 흥미롭고 공감되는 제목
- 예: "진정한 평안을 찾아서", "평일을 견디는 힘, 인내와 소망"

#### (2) 성경 본문 소개 (H2)
- 대표 구절 1-2개를 인용 (블록쿼트 사용)
- 구절 뒤에 (책 장:절) 표기
- 배경 및 맥락 설명 (2-3문단)
- 일상적 질문으로 시작하여 독자 공감 유도

#### (3) 묵상과 해석 (H2)
- 성경 본문의 의미
- 오늘날 우리에게 주는 교훈
- **반드시 3-4개의 소주제(H3)로 나누어 설명**
- 각 소주제는 구체적이고 실천적인 제목으로

#### (4) 찬송가가 전하는 메시지 (H2)
- H3 제목: "찬송가 [번호]장: [제목]" 형식 (1개만)
- **⚠️ 시즌별 찬송가 규칙**:
  - 성탄 찬송가 (주제: "성탄"): 12월에만 사용 (현재: {{current_month}}월)
  - 부활절 찬송가 (주제: "부활"): 3-4월에만 사용
  - 시즌이 맞지 않으면 다음 찬송가 선택 또는 섹션 생략
- 제목 다음 줄에:
  - 전체 가사 보기 링크 (있는 경우)
  - 유튜브 듣기 링크
  - 유튜브 iframe 임베드 (비디오 ID 있는 경우만)
- 가사 일부 (1절 또는 후렴) 코드블록으로 인용
- 찬송가의 의미와 메시지 설명 (2-3문단)

#### (5) 오늘의 적용 (H2)
- 구체적인 실천 방법 3가지
- 번호 목록(1, 2, 3)으로 작성
- 각 항목: **첫 문장 굵게** + 자세한 설명 (2-3줄)

#### (6) 마무리 기도 (H2)
- 2-3문단의 기도문
- 주님께 드리는 기도 형식 (존칭 사용)
- "예수님의 이름으로 기도합니다. 아멘." 으로 마무리

#### (7) 참고 (구분선 후 - H2 아님!)
- **오늘의 찬송**: [번호]장 (링크들)

### 2. 작성 스타일

✅ **해야 할 것**:
- 따뜻하고 진솔한 톤
- 일상 언어 사용 (쉽고 공감되게)
- 구체적인 예시와 비유
- 성경 구절 정확하게 인용
- 분량: 1500-2500자

❌ **하지 말아야 할 것**:
- 이모지 사용 금지
- 복잡한 신학 용어
- 추상적이고 관념적인 설명
- 과장이나 허위 정보
- 제공되지 않은 내용 임의 추가

## 🎯 키워드별 작성 가이드

### 날짜/요일별 고려사항
- 주일(일요일): 예배, 찬양, 은혜, 거룩
- 월요일: 새로운 시작, 소망, 용기, 힘
- 주중(화-목): 인내, 일상의 신앙, 직장생활, 관계
- 금요일: 감사, 안식 준비, 정리
- 토요일: 안식, 쉼, 회복, 묵상

### 교회 절기 (해당 시기일 경우)
- 대림절(11-12월): 기다림, 소망, 예수 탄생
- 성탄절(12월 25일): 성탄, 사랑, 선물, 구원
- 사순절(부활절 40일 전): 회개, 금식, 십자가
- 부활절(3-4월): 부활, 새 생명, 승리
- 추수감사절(11월): 감사, 수확, 축복

### 계절별 특성
- 봄(3-5월): 새로움, 생명, 시작
- 여름(6-8월): 성장, 열매, 활력
- 가을(9-11월): 수확, 감사, 성숙
- 겨울(12-2월): 인내, 소망, 준비

## ⚠️ 중요 주의사항

1. **JSON 형식 준수**: 반드시 유효한 JSON으로 출력
2. **개행문자 이스케이프**: content 내부의 모든 줄바꿈은 `\n`으로 표현
3. **따옴표 이스케이프**: content 내부의 따옴표는 `\"`로 이스케이프
4. **찬송가 시즌 체크**: 현재 월({{current_month}})과 찬송가 주제 확인
5. **제공된 데이터만 사용**: 임의로 성경 구절이나 찬송가 추가 금지

이제 위의 데이터를 기반으로 {{keyword}}에 대한 신앙 묵상 블로그를 JSON 형식으로 작성해주세요.
